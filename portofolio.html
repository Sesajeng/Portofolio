Proyek Terbaru

Website - Mabook

Mabook (Ma'arif Book) adalah aplikasi perpustakaan berbasis web dengan dua role: siswa dan admin. Siswa dapat login, mendaftar, meminjam dan mengembalikan buku, serta melihat status peminjaman dan denda. Admin dapat mengelola buku, kategori, peminjaman, pengembalian, dan pengguna.

#laravel#php#bootstrap#HTML#laragon#MySql

Website - Fitur Audio Feedback

Fitur Audio Feedback merupakan pengembangan pada website CSIRT yang bertujuan untuk meningkatkan aksesibilitas bagi penyandang tuna netra. Fitur ini memberikan umpan balik suara saat pengguna berinteraksi dengan elemen penting pada website, seperti tombol dan navigasi. Teknologi yang digunakan dalam pengembangan fitur ini adalah JavaScript dan CSS.

#javascript#css

Mobile App - PWTEVENT

PWT EVENT adalah aplikasi mobile pemesanan tiket online untuk berbagai event di daerah Purwokerto. Aplikasi ini memiliki dua role: pengunjung dan admin. Pengunjung dapat login, melihat daftar event, membeli tiket, dan memantau histori pembelian. Admin memiliki akses penuh untuk mengelola event dan transaksi tiket.

#angular#ionic#laragon
